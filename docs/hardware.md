# 硬件连接指南

本文档详细说明 ESP32-P4 与 MIPI LCD 面板的硬件连接方式。

## 引脚定义

### ESP32-P4 DSI 引脚

ESP32-P4 集成 MIPI DSI 接口，使用专用引脚：

| 功能 | GPIO | 描述 |
|------|------|------|
| DSI_CLK_P | GPIO 40 | DSI 时钟正 |
| DSI_CLK_N | GPIO 41 | DSI 时钟负 |
| DSI_DATA_P0 | GPIO 42 | DSI 数据 0 正 |
| DSI_DATA_N0 | GPIO 43 | DSI 数据 0 负 |
| DSI_DATA_P1 | GPIO 44 | DSI 数据 1 正 |
| DSI_DATA_N1 | GPIO 45 | DSI 数据 1 负 |
| DSI_DATA_P2 | GPIO 46 | DSI 数据 2 正 |
| DSI_DATA_N2 | GPIO 47 | DSI 数据 2 负 |
| DSI_DATA_P3 | GPIO 48 | DSI 数据 3 正 |
| DSI_DATA_N3 | GPIO 49 | DSI 数据 3 负 |

### 控制引脚

| 功能 | GPIO | 描述 |
|------|------|------|
| LCD_RST | GPIO 21 | 液晶复位 (可选) |
| LCD_BLK | GPIO 40 | 背光控制 |
| TP_SDA | GPIO 17 | 触摸 I2C 数据 |
| TP_SCL | GPIO 18 | 触摸 I2C 时钟 |
| TP_RST | GPIO 20 | 触摸复位 |

## 电源要求

### 推荐电源配置

| 电压 | 用途 | 电流要求 |
|------|------|----------|
| 3.3V | ESP32-P4 核心 | 500mA |
| 1.8V | MIPI DSI | 100mA |
| 5V/12V | LCD 背光 | 根据面板 |

### 注意事项

1. **MIPI 信号完整性**
   - 使用差分走线，保持阻抗 100Ω ± 10%
   - 差分对内长度匹配 < 2.54mm
   - 避免信号跨越分割平面

2. **电源滤波**
   - 每个电源引脚添加 100nF + 10μF 电容
   - 靠近引脚放置

3. **ESD 保护**
   - 推荐添加 ESD 保护二极管
   - 特别是 I2C 和控制信号

## 常见问题

### Q1: 屏幕不显示

1. 检查电源是否正常
2. 确认 MIPI 引脚连接正确
3. 检查 DSI 时钟频率设置

### Q2: 触摸无响应

1. 确认 I2C 地址正确 (默认 0x5D)
2. 检查 SDA/SCL 上拉电阻
3. 验证复位时序

### Q3: 显示颜色异常

1. 检查像素格式设置 (16bit/24bit)
2. 确认 color byte order
3. 检查初始化命令
